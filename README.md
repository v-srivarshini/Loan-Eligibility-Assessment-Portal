# Loan Eligibility Assessment Portal

## Overview

A full-stack web application that allows users to check their loan eligibility based on predefined criteria.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Axios
- Mongoose

## Features

- Loan application form
- Eligibility assessment
- MongoDB data storage
- REST API integration
- Responsive user interface
- Popup result display

## Eligibility Criteria

- Age: 21–60 years
- Employment Status: Employed
- Monthly Income: Minimum ₹30,000
- Loan Amount: Maximum 60 × Monthly Income

## How to Run

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
